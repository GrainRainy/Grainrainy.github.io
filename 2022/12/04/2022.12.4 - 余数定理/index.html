余数加法定理

(a + b) % c = (a % c + b % c) % c

证明：

若 a = xc + p, b = yc + q
则 a % c = p, b % c = q

则有:
(a + b) % c = [(x + y)c + p + q] % c = (p + q) % c
(a % c + b % c) % c = (p + q) % c
